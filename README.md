# electron-experiments

[![Greenkeeper badge](https://badges.greenkeeper.io/lipis/electron-experiments.svg)](https://greenkeeper.io/)

## Install

```shell
$ cd /path/to/electron-experiments
$ npm install
```

## Run

```shell
$ npm start
```

## Build

```shell
$ grunt build-win
```

## Distribute

```shell
$ grunt dist-win
```
